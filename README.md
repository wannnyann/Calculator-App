# Calculator-App

## 🛠️使用技術
- Java
- IntelliJ IDEA
- GitHub

## 📋アプリ説明
- 足し算、引き算、掛け算、割り算、余剰の計算が出来るアプリです。
- ユーザーが都度、数字や演算子を入力して計算ができるようになっています。
- 無効な演算子を入力すると、無効な演算子です😹と表示されます。
- また、何度でも計算ができるようにwhile文を使用しています。

## 🔢このアプリで学べたこと
- 四則演算など(＋、−、×、÷、%)
- オブジェクト指向(基本)
- while文
- Scanner
- switch文

## 🚀 今後追加したい機能
- 無効な演算子のときに再入力させる
- 0 で割ったときのエラー処理
- メニュー方式の UI
